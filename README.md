# POPCRUSH PROJECT 2020
A free web collection of music from around the world.

## Composition
This music collection contains songs that are discovered on the period from the late 2019 to present (median year 2020). Added NEW artists, previously unmentioned or unheard of: 2 Chainz (factually unmentioned in Ariana Grande - Rule The World song), De Hofnar, James Foxx (including his song featuring T-PAIN), Kodak Black (previously unmentioned in T-PAIN - Florida Boy song), Lost Years, Mr. Talkbox, Tones and I, Travis Scott (factually mentioned in Travis Scott - Pick Up The Phone song), Victoria Justice, YoungBoy Never Broke Again, etc.
